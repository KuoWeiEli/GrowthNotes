#### 系列文

筆者在這一系列將嘗試使用大家耳熟能詳的寶可夢來介紹設計模式（Design Pattern）。
每篇文章可分為五部分，分別如下：
1. Introduce：介紹 Pattern
2. Main：尋找在寶可夢中的設計模式
3. UML：簡單介紹各個類別與其相依關係
4. Code：詳細介紹各個類別
5. Sample：程式碼演示
